ErrorWithResponse

open class ErrorWithResponse : Exception, Parcelable

Error container returned when the Braintree server returns a 422 Unprocessible Entity. A 422 occurs when a request is properly formed, but the server was unable to take the requested action due to bad user data. ErrorWithResponse parses the server's error response and exposes the errors.
