GetReturnLinkTypeUseCase

@RestrictTo(value = [RestrictTo.Scope.LIBRARY_GROUP])
class GetReturnLinkTypeUseCase(merchantRepository: MerchantRepository)

Use case that returns a return link type that will be used for navigating from App Switch or browser back into the merchant app.

If a user unchecks the "Open supported links" checkbox in the Android OS settings for the merchant's app. If this setting is unchecked, this use case will return ReturnLinkTypeResult.DEEP_LINK, otherwise ReturnLinkTypeResult.APP_LINK will be returned.
