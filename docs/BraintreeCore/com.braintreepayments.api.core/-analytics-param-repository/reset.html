reset

fun reset()

Resets the sessionId and clears all other repository values.

Note that this function is called in different spots of the SDK lifecycle for different payment modules. Some modules call reset during launch of the SDK. The PayPal module calls reset at the end of the payment flow to persist the sessionId value set from the Shopper Insights module.
