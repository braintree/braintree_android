data class ShopperInsightsInfo(val isEligibleInPayPalNetwork: Boolean, val isPayPalRecommended: Boolean, val isVenmoRecommended: Boolean)

Data class encapsulating the result of a shopper insight api request.

This class holds information about the recommended payment methods for a shopper The recommendations include flags for whether payment methods like PayPal or Venmo should be displayed with high priority in the user interface.
