parseBrowserSwitchResult

@Nullable
open fun parseBrowserSwitchResult(@NonNull context: Context, @Nullable intent: Intent): BrowserSwitchResult

After calling startPayment, call this method in your Activity or Fragment's onResume() method to see if a response was provided through deep linking.

If a BrowserSwitchResult exists, call onBrowserSwitchResult, to allow the SDK to continue tokenization of the PayPalAccount.

Make sure to call clearActiveBrowserSwitchRequests after successfully parsing a BrowserSwitchResult to guard against multiple invocations of browser switch event handling.

Return

BrowserSwitchResult when a result has been parsed successfully from a deep link; null when an input Intent is null

Parameters

context

The context used to check for pending browser switch requests

intent

The intent containing a potential deep link response. May be null.
